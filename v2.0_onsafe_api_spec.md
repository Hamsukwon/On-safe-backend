## 📷 3. 카메라 (Camera)

### `POST /api/camera/stream` — 프레임 전송 + AI 추론 ⭐ 핵심

**Request** — `multipart/form-data`

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `frame` | file (JPEG) | ✅ | 카메라 프레임 이미지 |
| `user_id` | string | ✅ | 사용자 ID |
| `device_id` | string | ✅ | 기기 ID |

**Response 200**
```json
{
  "status": "ok",
  "score": 0.82,        // 낙상 위험 점수 0.0~1.0
  "fall": true,         // 낙상 감지 여부 (is_fall ❌)
  "log_id": "abc123"    // fall=true 시 Firestore log_id 반환
}
```

**처리 흐름**
```
1. JPEG → MediaPipe 포즈 추출 (33 관절 좌표)
2. 관절 각도 / 각속도 / 가속도 / 중심 동역학 피처 계산
3. Savitzky-Golay 필터 노이즈 제거
4. Redis LPUSH → LTRIM(150) → EXPIRE(10s)
5. 버퍼 ≥ 30행이면 pkl 모델로 AI 추론
6. fall=True → Firestore fall_logs 저장 + FCM 전송
```

---
